# Data Analyst Portfolio — Alfian Afriansyah

A portfolio of end-to-end data analytics case studies built with SQL (BigQuery), Python, and Looker Studio. 
Each project covers the full analytics process, from defining the business problem and performing data analysis to generating actionable insights and business recommendations.
